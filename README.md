# CLICKER IDLE

## Game Description:
CLICKER-IDLE is a browser-based idle clicker game built with vanilla JavaScript where players generate money, unlock avatars, purchase upgrades, activate powerful skills, collect Karu Gems, unlock music tracks, and customize themes while progressing through achievements and automation systems.

The game starts simple — clicking a button to make money — but gradually evolves into a full idle experience with autoclickers, progression mechanics, unlockable content, and character-based abilities.

As players progress, they can unlock unique avatars such as Karu, Kazzy, Ricardo, Spinal, and Robin, each tied to special upgrade paths and skill systems.



# Gameplay

The player earns money by clicking the **Make Money** button.

Money can be spent on:

* Autoclickers
* Click Power upgrades
* New Avatars
* Skills and unlockables

Autoclickers automatically generate money over time, allowing the player to progress even faster.

The game also includes:

* Achievement unlocks
* Karu Gem collection system
* Theme switching
* Background music unlocks
* Save & Load system
* Avatar progression
* Skill activation system

The objective is to fully unlock all avatars, achievements, skills, and music tracks while building the highest amount of money possible.


# Core Systems

## Money System

Players generate money manually and automatically using autoclickers.

## Autoclickers

Autoclickers continuously generate passive income over time.

## Click Power

Upgrading click power increases the amount of money earned per click.

## Karu Gems

Special currency earned by:

* Clicking milestones
* Unlocking achievements
* Time-based generation

Karu Gems are used to activate powerful character skills.

## Avatar System

Unlockable avatars include:

* Karu
* Kazzy
* Ricardo
* Spinal
* Robin

Each avatar includes unique flavor text and associated skills.

## Skill System

Each character has a unique ability:

* Increased autoclicker efficiency
* Click power multipliers
* Massive stat boosts
* Special unlock rewards

## Achievement System

Achievements reward players with:

* Karu Gems
* Music unlocks
* Progress milestones

## Music System

Players can unlock and play multiple background music tracks throughout progression.

## Theme System

The game includes multiple visual themes with:

* Dynamic backgrounds
* UI recolors
* Alternate assets
* Styled console appearance

# Game Mechanics

## Progression

Players start with:

* 0 money
* 0 autoclickers
* Basic click power

As the game progresses:

* Upgrade costs increase
* More avatars unlock
* Passive income grows
* Achievements reward bonuses

## Automation

Autoclickers generate money continuously every game tick.

## Persistence

Player progress is stored locally using browser LocalStorage.
